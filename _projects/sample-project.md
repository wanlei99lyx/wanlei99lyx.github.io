## 项目简介

这是一个示例开源项目，展示了如何使用 AI 技术解决实际问题。

## 特性

- **特性一**: 简洁高效的 API 设计
- **特性二**: 完善的文档和示例
- **特性三**: 活跃的社区支持

## 技术栈

- Python 3.10+
- PyTorch
- FastAPI

## 快速开始

```bash
git clone https://github.com/Wlei/example-project.git
cd example-project
pip install -r requirements.txt
python main.py
```
