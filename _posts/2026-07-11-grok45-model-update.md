2026 年 7 月 9 日，马斯克的 xAI（现已并入 SpaceX，合称 SpaceXAI）正式发布了 **Grok 4.5**——这是被马斯克称为"Opus 级"的旗舰模型，也是 xAI 与 Cursor 联合训练的首个编程智能体模型。

相比 GPT-5.6 和 Claude Fable 5 的"更强"叙事，Grok 4.5 打出的牌是：**"90% 的能力，10% 的价格"。**

![](/assets/images/posts/grok45-brain.jpg)

## Grok 4.5 核心规格

| 指标 | 详情 |
|------|------|
| 架构 | MoE（混合专家模型） |
| 参数量 | **1.5T**（前代 Grok 4.3 的 3 倍） |
| 上下文 | **50 万 Token**（下周将推 100 万版本） |
| 训练硬件 | 数万张 NVIDIA GB300 GPU |
| 联合训练方 | **Cursor**（马斯克以约 600 亿美元收购） |
| 支持模态 | 文本 + 视觉输入 |
| 公司实体 | xAI 并入 SpaceX → **SpaceXAI** |

## 性能定位：Opus 级，但不争第一

Grok 4.5 在多项编程和工程基准上表现强劲，但马斯克本人的定位很务实——**不求全面第一，但求足够好用且便宜**。

其编程能力基本追平 Opus 4.7/4.8，在 **DeepSWE Bench 1.0** 等测试中甚至超越 Opus 4.8，法律 Agent 测试排名第一。

但在纯粹的"智商"指标（如 FrontierMath、复杂 SWE 工程）上，仍落后于 Claude Fable 5 和 GPT-5.6 Sol。这也符合马斯克的策略：**覆盖 90% 的场景，剩下的 10% 留给下一代。**

## 最大的杀器：价格

这是 Grok 4.5 最值得关注的数据。

| 模型 | 输入价格（/百万 Token） | 输出价格（/百万 Token） |
|------|----------------------|----------------------|
| **Grok 4.5** | **$2** | **$6** |
| Anthropic Opus 4.7 | $5 | $25 |
| GPT-5.6 Sol | $5 | $30 |
| Claude Fable 5 | — | ~$15+ |

Grok 4.5 的 **输出价格仅为 Opus 的 1/4，GPT-5.6 Sol 的 1/5**。更关键的是它的 **Token 效率**——在 SWE Bench Pro 上，Grok 4.5 完成任务仅用 **1.6 万 Token**，而 Opus 4.8 用了 **6.7 万**。

单任务成本对比：Grok 4.5 约 **$1.51**，Fable 5 Max 约 **$17.32**——差了一个数量级。

![](/assets/images/posts/grok45-compare.jpg)

## 产品整合：Cursor 不再独立

收购 Cursor 后，Grok 4.5 成为首个与 Cursor 深度绑定的模型。模式类似 OpenAI 把 Codex 并入 ChatGPT——**AI 聊天 + 编程 IDE 在走向融合**。

Cursor 内部可直接调用 Grok 4.5 进行代码生成、审查和重构。SpaceXAI 的策略是：用 Grok 4.5 的低价优势吸引开发者使用 Cursor，形成"模型 → IDE → 开发者生态"的闭环。

## 应用场景

Grok 4.5 重点覆盖的场景非常明确：

- **编程开发**：大型代码库、多仓库协作、Rust/C++ 等挑战性任务
- **法律**：合同分析、法律 Agent（当前排名第一）
- **金融**：金融分析、Excel 建模
- **网络安全**：漏洞识别与修复

这些都是**企业愿意付费**的场景，也是 Grok 4.5"够用又便宜"策略最能发力的地方。

## 我的看法

Grok 4.5 的发布标志着 AI 模型竞争的第三阶段：

1. **阶段一（2023-2024）**：比谁更强——GPT-4 vs Claude 3
2. **阶段二（2025）**：比谁更全——多模态、长上下文、Agent
3. **阶段三（2026）**：**比谁更省**——GPT-5.6 用 Sol 打高端、Luna 打低成本，Grok 4.5 用极致的性价比切市场

马斯克的策略非常清晰：**不争第一，争最大。** 用 Opus 级别 90% 的能力、十分之一的价格，把企业市场的长尾需求吃掉。当一个模型"够好又便宜"时，企业没有理由不用。

接下来几个月的"性价比之战"会非常精彩。
